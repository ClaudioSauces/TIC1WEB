5. Actividad. Códigos y nombres HTML

Dados los siguientes caracteres a continuación escribe su equivalente mediante el nombre HTML en un documento HTML válido. En caso de no existir un nombre HTML usa el número HTML.

; < > & ' " \ backslash / / © copyright,   non-breaking space, $ $

Existen 2 formas de codificar los caracteres.

• La primera y más fácil de recordar es mediante el nombre HTML: &nombre; • Si no hay más remedio entonces usamos el código HTML: &#numero; Podéis usar el recurso www.ascii.cl
